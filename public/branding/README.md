# Mantix Branding Assets

Ubicacion oficial para los PNG del branding dentro de la app.

## Archivos esperados

- `mantix-logo-full-light.png`
- `mantix-logo-full-dark.png`
- `mantix-mark-light.png`
- `mantix-mark-dark.png`

## Uso sugerido

- `mantix-logo-full-light.png`: sidebar y superficies claras.
- `mantix-logo-full-dark.png`: sidebar y superficies oscuras.
- `mantix-mark-light.png`: favicon base, usos compactos claros, mobile.
- `mantix-mark-dark.png`: usos compactos oscuros, mobile y shortcuts.

## Tamanos recomendados

- Logo horizontal principal: `1200 x 300 px` PNG transparente.
- Version sidebar compacta: `240 x 64 px` PNG transparente.
- Isotipo / cuadrado: `512 x 512 px` PNG transparente.
- Favicon base: `64 x 64 px` y `32 x 32 px`.
